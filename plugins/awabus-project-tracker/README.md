# AwaBus Project Tracker

A repo-local Codex plugin for tracking the AwaBus project across backend, mobile apps, PWA, admin dashboard, IVR, testing, deployment, and academic deliverables.

## Included Skill

- `track-awabus-project`: turns the project brief, repo state, and user updates into a practical project tracker with milestones, owners, blockers, risks, decisions, and next actions.

## Starter Prompts

- Track AwaBus project progress.
- Create an AwaBus sprint plan.
- Summarize AwaBus risks and blockers.
